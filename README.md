# Customer Data Analysis for Business Insights

## Problem Statement
Analyze customer data to understand behavior and improve customer retention.

## Tools Used
- Python
- SQL
- Power BI

## Project Overview
This project focuses on analyzing customer demographics and transaction data to uncover behavior patterns, segment customers, and generate actionable business insights.

## Outcomes
- Queried and analyzed customer datasets using SQL.
- Segmented customers based on behavior and purchase patterns.
- Presented insights through Power BI dashboards to support marketing strategies.

## Repository Structure
- data/: Raw customer and transaction datasets
- sql/: SQL queries used for analysis
- notebooks/: Python-based EDA
- powerbi/: Power BI dashboard file
- visuals/: Dashboard screenshots
- reports/: Final business insights
